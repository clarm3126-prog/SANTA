# SANTA (산타)

안전 기반 등산 동행 플랫폼 웹 프로토타입입니다.

## 현재 구조

- `index.html`: 화면 마크업과 정보 구조
- `assets/css/styles.css`: 전체 스타일
- `assets/js/app.js`: 화면 전환, 폼 검증, 모달, 토스트 등 인터랙션

## 제품 방향

이 프로토타입은 아래 세 축을 검증하기 위한 MVP입니다.

1. 정확한 산 탐색과 구조화된 후기
2. 소개팅이 아닌 안전한 그룹 동행 매칭
3. 실명 인증, 위치 공유, 귀가 체크 중심의 신뢰 UX

## 실행 방법

정적 파일이라서 브라우저에서 `index.html`을 열면 됩니다.

## 다음 우선순위

1. 더미 데이터를 배열/객체로 분리해 렌더링 구조 만들기
2. 디자인 토큰과 컴포넌트 단위로 CSS 정리하기
3. React 또는 Next.js로 옮겨 상태 관리와 라우팅 붙이기
4. 지도 SDK, 인증, 후기 데이터 모델 설계 시작하기
